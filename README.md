# MyNAS - Premium Cloud Storage

A premium, private cloud storage system (NAS) with a beautiful React UI.

## Features

- **Modern UI**: Built with React, Vite, and Framer Motion.
- **Glassmorphism Design**: Sleek dark mode interface.
- **File Management**: Upload, create folders, delete, and download.
- **Preview**: Icons for various file types.
- **Responsive**: Works on desktop and mobile.

## Installation

1. Open a terminal in this folder.
2. Run `npm install` to install dependencies.
3. Run `npm run build` to build the frontend.

## Running the NAS

Double-click `start_nas.bat` to start the server.

Or via terminal:
```bash
npm start
```

Access at `http://localhost:3000`.

## Development

To run in development mode (with hot reload):
1. Run `npm install` in root.
2. Run `npm install` in `frontend` folder.
3. Run `npm run dev` in root (requires `concurrently`).

Enjoy!
